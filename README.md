# Compound Chat AI

**Privacy-first, zero-cost-to-launch AI chatbot SaaS platform**

Embed a powerful AI chatbot on your website with all knowledge stored securely in your Google Drive.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## 📋 Tech Stack

- Next.js 15 + TypeScript + Tailwind CSS (#0af theme)
- Supabase (Auth, Database)
- Groq + DeepSeek LLMs
- Lightweight embeddable widget

## 🎯 Key Features

- Landing page with pricing
- User authentication (login/signup)
- Dashboard (widgets, analytics, settings)
- Embeddable AI chatbot widget
- Google Drive integration
- 30 lifetime conversations (free tier)
- Webhook integrations

## 📁 Structure

```
src/app/
├── page.tsx           # Landing page
├── login/

    ├── widgets/
```
